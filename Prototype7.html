Le Projet JL - Qu'est-ce que c'est?

Le projet JL est une solution innovante conçue par Jean-Louis LIKULA pour résoudre un problème simple mais crucial : la conservation et l'organisation des liens web.


Mes motivations
  • L'infinité du monde web - le web représente l'infini dans ce monde, avec un nombre incalculable de ressources à découvrir.
  • Tout passe par un chemin : un lien web.
  • J'ai découvert tant de merveilles grâce à de simples liens, qui ne sont qu'une forme de texte court mais qui ouvrent des portes immenses.
  • J'ai aussi perdu tant de ces merveilles parce que je n'avais pas d'endroit pour les stocker efficacement.
Le problème

Les gestionnaires de fichiers classiques :

  • Servent principalement à stocker des vidéos, images, et d'autres types de fichiers accessibles nativement
  • Sont limités en termes de capacité de stockage
  • Sont souvent restreints à un appareil ou à un support de stockage spécifique
  • Ne sont pas optimisés pour organiser et accéder rapidement à des liens web
La solution

J'ai créé un système qui est :

  • Toujours accessible (n'importe où, n'importe quand)
  • Non limité, indépendant d'un système spécifique (base de données, stockage, connexion internet)
  • En harmonie avec l'infinité du web
  • Accessible hors-ligne
  • Portable et adaptable

Une solution où l'utilisateur est maître de tout, modelant son propre monde numérique sans limites.

Comment utiliser ce projet?

Voici comment tirer le meilleur parti de cette application :


Premiers pas
  1. Commencez par ajouter votre premier lien ou créer votre premier dossier
  2. Organisez vos liens selon vos préférences
  3. Ajoutez du texte pour contextualiser vos liens ou noter des informations importantes
  4. Important : N'oubliez pas d'exporter régulièrement un fichier JSON pour sauvegarder votre travail

Sauvegarde et restauration

Le format des sauvegardes suit la convention :

backup_YYYY-MM-DD_HH-MM-SS.json

Par exemple : backup_2025-11-01_10-30-45.json

Pour restaurer vos données :

  1. Allez dans l'explorateur
  2. Cliquez sur l'icône d'importation
  3. Sélectionnez votre fichier de sauvegarde JSON

Vos données seront immédiatement restaurées dans l'application.

Préservation du Projet

Pour assurer la préservation et l'accessibilité permanente de votre projet, suivez ces conseils :

  • Uploadez régulièrement votre projet en ligne dans le cloud
  • Privilégiez les dépôts publics pour un accès permanent
  • Effectuez des mises à jour régulières, surtout pour les fichiers JSON qui contiennent vos données
  • Conservez une copie de secours dans plusieurs emplacements
Plateformes recommandées
GitHub
IPFS
Google Drive
Mega
Mediafire

GitHub est idéal pour les projets de code. Vous bénéficiez d'un contrôle de version, d'une collaboration facile et d'une disponibilité exceptionnelle. Parfait pour les développeurs.

Les plateformes IPFS (InterPlanetary File System) offrent un stockage décentralisé persistant. Vos données sont distribuées sur plusieurs nœuds, ce qui les rend résistantes à la censure et aux pannes.

Options recommandées : ipfs.io, web3.storage, nft.storage, pinata cloud.

Google Drive est facile à utiliser et offre 15GB d'espace gratuit. Il est accessible depuis n'importe quel appareil et propose des fonctionnalités de partage simples.

Mega offre 20GB gratuits avec un chiffrement de bout en bout. C'est une bonne option pour la confidentialité et le stockage sécurisé.

MediaFire est simple et rapide avec 10GB gratuits. Il est idéal pour le partage de fichiers et l'accessibilité multi-plateformes.

Générer un QR code d'accès

Entrez l'URL où votre projet est stocké pour générer un QR code d'accès rapide.

Scannez ce code pour accéder à votre projet

Ce QR code est votre clé d'accès universelle. Même si vous oubliez vos identifiants ou perdez l'accès à votre compte, ce code vous permettra de retrouver votre projet. Conservez-le précieusement, comme une carte d'identité numérique pour votre projet.
