À propos du Projet JL

Le projet JL est un concept inventé par Jean-Louis LIKULA pour résoudre un problème fondamental : comment préserver et organiser efficacement les ressources web infinies que nous découvrons chaque jour.

Mes motivations
  • L'infinité du monde web, selon moi, représente l'infini dans ce monde
  • Il existe un nombre incalculable de ressources à découvrir, à explorer
  • Chaque ressource est accessible via un simple lien
  • J'ai découvert d'innombrables merveilles grâce à de simples URLs
  • Mais j'en ai aussi perdu beaucoup, faute d'un système adéquat pour les conserver
La solution recherchée

Je devais créer un système qui serait à la fois :

  • Toujours accessible (n'importe où, n'importe quand)
  • Non limité, indépendant d'un système spécifique
  • Hors-ligne quand nécessaire
  • Contrôlé entièrement par l'utilisateur
  • Sans limites, à l'image de l'infinité du web
Comment utiliser ce projet

Voici comment tirer le meilleur parti de Projet.JL pour organiser vos ressources web :

Commencer avec Projet.JL
  1. Créez votre premier dossier ou ajoutez votre premier lien
  2. Explorez l'interface intuitive et adaptative
  3. Organisez vos ressources selon vos besoins
Sauvegarde et portabilité

Important : Pour conserver vos données, vous devez exporter régulièrement un fichier JSON.

Ce fichier agit comme votre "base de données" et vous permet de retrouver toutes vos données sur n'importe quel appareil.

Structure du projet :
votre-projet
data
backup_2025-11-07_10-30-45.json
index.html
styles.css
scripts.js
Format des fichiers de sauvegarde

Les sauvegardes sont enregistrées au format :

backup_YYYY-MM-DD_HH-MM-SS.json

Par exemple : backup_2025-11-07_10-30-45.json

Ce format inclut la date et l'heure pour vous aider à identifier facilement vos sauvegardes.

Sécurité et Accessibilité

Conseil de sécurité : Pour une meilleure préservation de votre projet, nous vous recommandons de :

  • Uploader régulièrement votre projet en ligne sur une plateforme de stockage cloud
  • Rendre votre projet public dans les plateformes où il est stocké pour y avoir toujours accès
  • Mettre à jour régulièrement vos fichiers JSON de sauvegarde
  • Conserver le QR code de votre projet dans un endroit sûr pour y accéder rapidement depuis n'importe quel appareil
Enregistrez votre projet en ligne

Conservez ce QR code précieusement. Il vous permettra d'accéder à votre projet depuis n'importe quel appareil, même sans vos identifiants.
