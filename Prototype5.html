Bienvenue sur Projet.JL

Une application de stockage ♾️ infini pour vos liens et documents.

À propos du Projet.JL

Le projet JL est un projet conçu par Jean-Louis LIKULA pour résoudre un problème spécifique : la conservation et l'organisation des liens web de manière efficace et pérenne.

Mes motivations

L'infinité du monde web est ce qui représente l'infini dans ce monde. Il y a tellement un nombre incalculable de ressources à découvrir, à explorer, et tout passe par un chemin : un lien web.

Vous n'imaginez pas combien de merveilles j'ai pu découvrir grâce à de simples liens, et combien de ces merveilles j'ai pu oublier ou perdre, parce que je n'avais pas un endroit où les stocker efficacement.

Les gestionnaires de fichiers classiques ne servent qu'à stocker des médias et sont toujours limités en nombre de stockage, à l'appareil, ou au support de stockage.

Ma solution

Je devais créer un système, trouver une solution qui serait à la fois :

- Toujours accessible (n'importe où, n'importe quand)
- Non limité, non dépendant d'un système (base de données, stockage, connexion internet)

Une solution qui concorde avec l'infinité du web, qui peut devenir le web à sa façon - pas besoin de dépendre d'une base de données, d'un système, d'une platefome, d'une connexion internet.

Voilà pourquoi j'ai créé ce système de stockage infini, accessible n'importe où, n'importe quand, hors-ligne, qui peut se trouver partout à la fois et où l'utilisateur est le maître de tout.

Comment utiliser ce projet ?

Voici quelques étapes simples pour commencer :

Première utilisation

1. Commencez par ajouter votre premier lien, créer votre premier dossier ou ajouter du texte.

2. Avant de quitter l'application, exportez un fichier JSON - ce sera votre base de données contenant tous vos liens et dossiers.

3. Exportez-le dans la racine de votre projet dans le dossier "data".

Utilisation ultérieure

Si un jour lors de vos activités futures l'application semble vide (peut-être parce que vous avez changé de navigateur ou vous utilisez un autre appareil), n'ayez crainte.

Vous n'aurez qu'à "importer" votre fichier JSON qui se trouve dans le dossier data à la racine de votre projet, et toutes vos données seront restaurées.

Préservez et partagez votre projet en toute sécurité.

Conseils de sécurité

Pour préserver votre projet en cas d'oubli de compte ou de mot de passe, nous vous recommandons de :

  • Toujours uploader votre projet en ligne dans un service cloud
  • Rendre votre projet public dans les endroits où il est stocké pour y avoir toujours accès
  • Faire des mises à jour régulières, surtout des fichiers JSON qui représentent votre base de données

Plateformes recommandées

GitHub

GitHub est idéal pour les projets web comme celui-ci. Il offre un versionnement du code et un accès facile depuis n'importe quel navigateur. Vous pouvez également activer GitHub Pages pour accéder directement à votre application en ligne.

Plateformes IPFS

Les plateformes IPFS (InterPlanetary File System) comme IPFS.io, Web3.storage, NFT.storage ou Pinata Cloud permettent de stocker vos fichiers de manière décentralisée et immuable. Vos fichiers sont accessibles via un hash unique, garantissant que vos données resteront toujours accessibles tant que quelqu'un les partage sur le réseau.

Google Drive

Google Drive est une solution de stockage cloud populaire et fiable avec un espace gratuit généreux. Vous pouvez partager votre projet avec un lien public pour y accéder facilement.

Mega / Mediafire

Ces plateformes offrent un stockage généreux et permettent de partager facilement des liens publics vers vos fichiers, même si vous n'êtes pas connecté à votre compte.
